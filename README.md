# redeon.sh

Criado em Shell Script para uso exclusivo no Kali Linux, esta ferramente tem a função de restabelecer sua conexão sem que precise reiniciar seu sistemas.
para iniciar de a permissão de execução:

chmod 777 redeon.sh

depois:
./redeon.sh

ou para ver ajuda:
./redeon --help
